
# Build Instruction

A Process Schedule Simulator using diffrent algorithms

## Getting Started

Ensure you have a JDK installed on your Linux system. You can check by running in  in Linux:
>```java -version```
otherwise install by running
>```sudo apt install default-jdk```
## Steps
 - navigate to the root of the **src** folder in the terminal
 - execute ```javac *.java``` to compile
 - execute ```java App``` to run program
